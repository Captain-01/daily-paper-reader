## 摘要
高通量微流控活细胞成像生成丰富的单细胞数据。然而，用于定位感兴趣区域（每个区域包含一个细胞群）并从记录图像中移除周围微流控结构的半自动化程序，其规模随感兴趣区域数量增加而扩大。这阻碍了实时图像分析，并将洞察时间延迟数小时至数天。我们引入了用于微流控培养芯片的设计感知和实时能力（DART）范式，该范式将CAD蓝图与物理芯片对齐，从而实现了与通量无关的所有感兴趣区域定位以及跨不同感兴趣区域几何形状和芯片布局的全自动化图像处理。DART通过嵌入式基准标记和基于深度学习的标记检测建立这种对齐。我们使用瑞士军刀芯片验证DART，该芯片在1164个感兴趣区域位置上组合了八种结构不同的感兴趣区域设计。DART在五分钟内定位所有感兴趣区域，在40毫秒内从原始显微镜图像中移除微流控结构，并在每张图像不到1.1秒内执行全自动图像分析，包括细胞分割。这些能力共同将DART确立为一种端到端的硬件-软件范式，具有实时分析能力，为闭环和结果驱动的智能显微镜铺平了道路。

## Abstract
High-throughput microfluidic live-cell imaging generates rich single-cell data. Yet semi-automated procedures for locating regions of interest (RoIs), each containing one cell population, and removing surrounding microfluidic structures from recorded images, scale with the number of RoIs. This prevents real-time image analysis and delays time-to-insight by hours to days. We introduce the Design-Aware and Real-Time capable (DART) paradigm for microfluidic cultivation chips, which aligns the CAD blueprint with the physical chip and thereby enables throughput-independent localization of all RoIs and fully automated image processing across diverse RoI geometries and chip layouts. DART establishes this alignment through embedded fiducial markers and deep-learning-based marker detection. We validate DART using the Swiss Army Knife chip, which combines eight structurally distinct RoI designs across 1164 RoI locations. DART localizes all RoIs in five minutes, removes microfluidic structures from raw microscopy images in 40 ms, and performs fully automated image analysis, including cell segmentation, in under 1.1 s per image. Together, these capabilities establish DART as an end-to-end hardware-software paradigm with real-time-capable analysis that paves the way toward closed-loop and outcome-driven smart microscopy.

### 2. 论文提出的方法论：核心思想、关键技术细节

论文提出的**DART（Design-Aware and Real-Time capable）**范式的核心思想是：**在芯片设计和制造阶段就嵌入结构信息（基准标记和ID），使得物理芯片与其CAD蓝图之间建立明确、可计算的空间对应关系。** 这种“设计感知”能力使得整个图像分析流程能够自动利用蓝图中的先验知识。

- **核心技术框架**：
    1.  **DART芯片设计**：在标准微流控芯片CAD设计中，为每个RoI添加两个基准标记（一个十字和一个圆圈）和一个唯一数字ID，这些标记位于RoI附近，位置关系在蓝图中已知。
    2.  **基准标记检测**：使用**YOLOv26-s**深度学习模型，在实时条件下高精度地检测显微镜图像中的十字和圆圈标记。该模型提供亚微米级的定位精度（95%百分位误差为0.18 μm）和毫秒级的推理速度（16.4 ms/张）。
    3.  **粗对齐**：通过手动拍摄三个参考RoI的图像，自动计算一个仿射变换，将整个CAD蓝图的RoI坐标映射到显微镜载物台的坐标系。这是一个固定的、不随RoI数量增加而扩展的操作，耗时约5分钟。
    4.  **细对齐**：对于每个记录的图像，检测基准标记，计算图像的旋转和平移，将其与蓝图精确对齐。然后，根据RoI ID自动加载对应的蓝图掩膜，从图像中精确地裁剪出并屏蔽掉微流控结构。
    5.  **细胞分割**：在预处理好的、仅包含培养区域的图像上，使用**Cellpose-SAM**进行细胞实例分割，并自动移除与掩膜重叠的错误阳性结果。

- **算法流程**（文字说明）：
    1.  **YOLO模型**检测图像中的十字和圆圈标记。
    2.  匹配标记对，并根据蓝图中的标准间距验证其有效性，计算旋转角度。
    3.  将图像旋转至蓝图坐标系。
    4.  根据蓝图中的RoI掩膜裁剪图像，只保留培养腔室内部区域。
    5.  在裁剪后的图像上运行Cellpose-SAM进行细胞分割。

### 8. 不足与局限

- **定位精度限制**：粗对齐后，90%的RoI定位误差达18.39 μm（约视野宽度的17.6%），可能导致培养腔室边缘区域成像不完全，对非常小的RoI影响较大。

- **计算资源需求**：虽然每张图像处理速度很快，但仍高度依赖GPU。一个标准的、不配备高端GPU的实验控制计算机可能无法达到论文中报告的实时处理速度。
- **未实现真正的“闭环”**：论文声称DART为闭环实验铺平了道路，但实验本身并未演示一个真正的闭环应用（如根据细胞密度自动调整培养基流速）。它目前仍然是一个独立的分析流水线。
- **缺乏定量准确性验证**：论文未定量验证DART分析结果（如细胞计数、生长速率）相对于人工手动分析或其他自动化方法的准确性，因此其生物学结论的准确性存在未知数。
